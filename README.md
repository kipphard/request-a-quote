# Angebotsanfrage — Request a Quote for WooCommerce

Let customers build a **quote list** and send a request instead of checking out — for B2B,
wholesale, and hidden-price shops. Add products to a quote, submit an inquiry form, and the
request lands in the WordPress admin and your inbox.

## ▶ Live demo (no install)

**[Open the demo in WordPress Playground](https://playground.wordpress.net/?blueprint-url=https://raw.githubusercontent.com/kipphard/angebotsanfrage/main/demo/blueprint.json)**
— a full WooCommerce shop with the plugin active and sample products, running in your browser.

## Install

Requires **WooCommerce**. Download the latest
[release](https://github.com/kipphard/angebotsanfrage/releases) zip → WordPress admin →
*Plugins → Add New → Upload Plugin → Activate*. Configure under the **Angebotsanfrage** menu;
place the `[angebotsanfrage]` shortcode on a page for the quote list + request form.

## Features (free)

- "Add to quote" button on shop and product pages (alongside or replacing add-to-cart)
- Session-based quote list + a request form (name, email, company, phone, message)
- Requests stored as a WordPress post type + emailed to you and the customer
- Optionally hide prices / the cart for selected products and request a quote instead
